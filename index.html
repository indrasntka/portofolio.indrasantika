About Me

Hallo semua! perkenalkan nama saya I Putu Indra Santika Sangging. Saya berasal dari Singaraja, Bali. Saya merupakan mahasiswa dari Universitas Pendidikan Ganesha Jurusan Teknik Informatika angkatan 2021.

Pada saat ini saya merupakan mahasiswa semester 4 di Program Studi Pendidikan Teknik Informatika. Riwayat pendidikan yang saya tempuh dimulai dari TK Lila Kumara yang terletek di Amlapura. Kemudian melanjutkan ke tingkat SDN 1 Amlapura, lalu melanjutkan ke-SMPN 2 Amlapura dan pada akhirnya saya pindah ke Singaraja di sekolah SMAN 4 Singaraja.

My Project 1

pada project kali ini, kami para semester 3 tepatnya kelas C diberikan tugas oleh bapak I Gede Mahendra Darmawiguna, S.Kom., M.Sc. Pada project ini kami diminta untuk membuat animasi object bergerak berupa ikan, peti harta karun dan rumput laut. setiap object memiliki gerak tersendiri, contohnya pada ikan yang memiliki gerak maju kedepan, mundur, naik dan turun sedangkan pada peti harta karun memiliki animasi berupa naik ke permukaan laut lalu turun kembali ke permukaan laut. yang terakhir pada rumput laut memiliki animasi gerak berupa bergoyang layaknya seperti rumput yang terhempas ombak.

My Project 2

pada project kali ini, kami para semester 3 tepatnya kelas C diberikan tugas oleh bapak I Gede Mahendra Darmawiguna, S.Kom., M.Sc. Pada project ini kami diminta untuk membuat sebuah game greenfot yaitu game freding frenzy dimana pada game ini terdapat sebuah ikan yang harus mengumpulkan poin untuk menyelesaikan game ini. Cara untuk menyelesaikan game ini yaitu kita sebagai ikan harus memakan sebanayak mungkin untuk meraih poit sesuai target.
